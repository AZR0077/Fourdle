# Four in One Wordle
This is an adaption of the very popular web game Wordle, this version is edited to include three, four, five, and six-letter variations of Wordle.

# Made in JavaScript and HTML

Contains seven files:
  - Assets folder
  - JS game app folder
  - JS code file
  - HTML website file
  - CSS style file
