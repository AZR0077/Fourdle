# Fourdle
This is a adaption of the very popular web game Wordle, this version is editied to use four-letter words instead of five-letter words.

# Made in JavaScript and HTML

Contains seven files:
  - Assets folder
  - JS game app folder
  - JS code file
  - HTML website file
  - CSS style file
